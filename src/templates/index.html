Welcome to PMQF: The Process Mining Question Forge

Whether you're diving into process analysis or seeking clarity on intricate questions, this tool aims to inspire and guide you. It is developed targeting process mining analysts and process owners as part of a research project. It brings an intuitive approach to inspire question design and question refinement and thus helps to streamline your scoping and requirements engineering efforts to lay the groundwork for successful project outcomes.

Background

The currently deployed version of PMQF builds on top of a large collection of process mining analysis questions that are gathered from practical projects, case studies, and tool vendors. Based on these questions, we have developed a categorization schema that allows to access the question collection in a structured way and supports different usage scenarios. However, PMQF can be extended with any question collection and categorization schema.

  • Keyword Search: This feature allows you to search for questions based on keywords.
  • Guided Question Design: This feature guides you through a set of selections and helps you to design the most relevant questions for your project.
  • Question Refinement: This feature allows you to reflect upon your (existing) questions based on on the classification schema and ensures a shared understanding between all project stakeholders. We advice to use this feature with your complete project team.
However, before continuing with one of these options, you are free to familiarze yourself with the classification schema and some examples to better understand it's dimensions.

In case you are starting your process mining project, we recommend you to start with the guided question design feature. In case you already have a specific use case in mind that you can nail down based on a keyword, you can also try using the keyword search. If you already have a concrete question in mind and would like to refine it or ensure a common understanding between you and your project team, you can use the question refinement feature.
